# 利用規約・プライバシーポリシーのテンプレート

管理画面の「サーバー設定」から、利用規約とプライバシーポリシーをMarkdown形式で設定できます。以下のテンプレートをコピーして、サーバーの実情に合わせて編集してください。

!!! warning "注意"
    以下はテンプレート例です。法的な助言ではありません。必要に応じて専門家に相談してください。

---

## 利用規約テンプレート

以下を管理画面の「利用規約 (Markdown)」欄にコピーしてください。

```markdown
# 利用規約

最終更新日: YYYY年MM月DD日

## 1. サービスの概要

本サービス「（サーバー名）」（以下「本サービス」）は、ActivityPubプロトコルに基づく分散型ソーシャルネットワーキングサービスです。

## 2. 利用条件

- 本サービスの利用には、アカウントの登録が必要です。
- 13歳未満の方は本サービスを利用できません。
- アカウント登録時に正確な情報を提供してください。
- アカウントの管理責任はユーザー本人にあります。

## 3. 禁止事項

以下の行為を禁止します。

- スパム行為、大量の自動投稿
- ハラスメント、脅迫、いじめ
- 著作権を侵害するコンテンツの投稿
- 違法なコンテンツの投稿・共有
- 他のユーザーへのなりすまし
- サービスの運営を妨害する行為
- 児童の性的搾取に関するコンテンツ

## 4. コンテンツについて

- ユーザーが投稿したコンテンツの権利はユーザーに帰属します。
- 運営者は、サービスの提供に必要な範囲でコンテンツを利用できます。
- 規約に違反するコンテンツは事前の通知なく削除される場合があります。

## 5. ActivityPub連合について

本サービスはActivityPubプロトコルを使用しています。

- 公開投稿はリモートサーバーに配信され、コピーが保存される場合があります。
- 一度配信された投稿をリモートサーバーから完全に削除することは保証できません。
- リモートサーバーのポリシーは本サービスの管理外です。

## 6. サービスの変更・終了

- 運営者は、事前の通知によりサービスの内容を変更できます。
- サービスの終了時は、可能な限り事前に告知します。
- サービス終了時には、ユーザーデータのエクスポート期間を設けます。

## 7. 免責事項

- 本サービスは現状のまま提供されます。
- 運営者は、サービスの中断・データの損失について責任を負いません。
- ユーザー間のトラブルについて、運営者は仲裁する義務を負いません。

## 8. 規約の変更

本規約は変更される場合があります。重要な変更については事前に告知します。

## 9. お問い合わせ

本規約に関するお問い合わせは（連絡先）までお願いします。
```

## プライバシーポリシーテンプレート

以下を管理画面の「プライバシーポリシー (Markdown)」欄にコピーしてください。

```markdown
# プライバシーポリシー

最終更新日: YYYY年MM月DD日

## 1. 収集する情報

本サービスでは、以下の個人情報を収集します。

| 情報 | 目的 |
|------|------|
| メールアドレス | アカウント管理、通知 |
| パスワード（ハッシュ化） | 認証 |
| IPアドレス | セキュリティ、不正利用防止 |
| 投稿内容 | サービスの提供 |
| アップロードファイル | サービスの提供 |
| プロフィール情報 | サービスの提供、公開表示 |

## 2. 情報の利用目的

収集した情報は以下の目的で利用します。

- サービスの提供と改善
- アカウントの管理と認証
- 不正利用の防止
- サービスに関する通知

## 3. ActivityPubによるデータ共有

本サービスはActivityPubプロトコルを使用した分散型サービスです。

- **公開投稿**: フォロワーがいるリモートサーバーに配信されます。
- **プロフィール情報**: リモートサーバーからの問い合わせに応じて提供されます。
- **ダイレクトメッセージ**: 宛先のサーバーに送信されます。内容の暗号化は保証されません。
- **リモートサーバーでのキャッシュ**: 配信された情報はリモートサーバーにキャッシュ・保存される場合があり、本サービスの管理外です。

## 4. Cookieの使用

本サービスでは、認証のためにセッションCookieを使用します。このCookieはログイン状態の維持に必要です。

## 5. データの保持期間

- アカウント情報: アカウント存続中
- 投稿内容: 削除されるまで
- IPアドレス: セキュリティログとして一定期間
- 削除されたコンテンツ: 速やかにサーバーから削除

## 6. ユーザーの権利

ユーザーは以下の権利を有します。

- **アクセス**: 自身の個人情報へのアクセス
- **訂正**: プロフィール情報の訂正
- **削除**: アカウントの削除によるデータの消去
- **エクスポート**: 自身のデータのエクスポート（設定画面から）

## 7. アカウント削除

アカウントを削除すると：

- アカウント情報とプロフィールが削除されます。
- 投稿がサーバーから削除されます。
- リモートサーバーに削除リクエストが送信されますが、完全な削除は保証されません。
- アップロードされたファイルが削除されます。

## 8. セキュリティ

パスワードのハッシュ化やHTTPS通信など、適切なセキュリティ対策を実施しています。

## 9. お問い合わせ

プライバシーに関するお問い合わせは（連絡先）までお願いします。
```

## 設定方法

1. 管理者アカウントでログインします。
2. 管理画面 (`/admin/settings`) を開きます。
3. 「利用規約 (Markdown)」欄にテンプレートを貼り付け、編集します。
4. 「プライバシーポリシー (Markdown)」欄にテンプレートを貼り付け、編集します。
5. 「保存」ボタンをクリックします。

設定したコンテンツは `/terms` と `/privacy` に公開されます。登録フォームにもリンクが表示されます。
